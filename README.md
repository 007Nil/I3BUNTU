# I3buntu
By executing this script you can install customised i3-wm in Ubuntu based system
<h1>Instructions---------</h1><br>
<p>
1 git clone https://github.com/007Nil/I3buntu.git <br>
2 cd I3buntu <br>
3 chmod +x i3buntu.sh <br>
4 ./i3buntu<br>
  </p>
